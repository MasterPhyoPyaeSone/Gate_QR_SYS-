
# GatePass QRSystem - GatePassDashboard


A complete Gate Pass Management System that uses QR codes for secure entry/exit validation and provides an admin dashboard for monitoring, logging, and managing user gate passes.


## Project Overview



The dashboard centralizes all activity, allowing administrators to monitor gate movements, track usage history, manage users, and generate reports. The system improves security, reduces errors, and increases efficiency for organizations that need accurate gate control.

## Run Locally

Clone the project

```bash
git clone https://github.com/MasterPhyoPyaeSone/Gate_QR_SYS-.git
```

Go to the project directory

```bash
  cd Gate_QR_SYS-

```

Install dependencies

```bash
  composer install
  npm install

```


Run migrations

```bash
  php artisan migrate --seed
```

Start the server

```bash
  npm run start
  or
  composer run dev
```
